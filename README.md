# 🍺 Belgian Beer Recommender

A fun, interactive quiz that recommends 3 real Belgian beers based on your mood and taste preferences.

## How It Works

1. **Age Gate** — Confirm you're of legal drinking age
2. **5 Questions** — Answer quick questions about your mood, bitterness tolerance, dessert preferences, strength preference, and flavor world
3. **3 Recommendations** — Get personalized Belgian beer picks with descriptions and fun facts

## Features

- 🍻 **20 real Belgian beers** — Trappists, lambics, saisons, abbey ales, witbiers, and more
- 🎯 **Taste matching** — Euclidean distance scoring across 5 flavor dimensions (bitter, sweet, fruity, strong, complexity)
- 🎨 **Belgian tavern theme** — Warm, cozy design with smooth transitions
- 📱 **Responsive** — Works on desktop and mobile
- ⚡ **Zero dependencies** — Vanilla HTML, CSS, and JavaScript

## Run Locally

Open `index.html` in your browser. That's it — no build step, no server required.
